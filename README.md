# Projects

#### Ghana Housing Insights 
[![Residential properties in Ghana...](images/Airport-Estate-768x432.jpg)](https://github.com/Ohenedarkoh/Ghana-housing-insights)

* **Built a full data pipeline** to scrape, clean, and load real-estate listings into **PostgreSQL** for property valuation analysis.
* **Developed and deployed an XGBoost prediction model** via a **FastAPI** service (Render) to provide real-time predictions.
    * *Performance Metrics:* $\text{R}^2=0.45$, $\text{RMSE}=0.7992$, $\text{MAE}=0.6058$ (Link to live docs: [https://ghana-housing-insights-1.onrender.com/docs](https://ghana-housing-insights-1.onrender.com/docs))
* **Generated a Power BI dashboard** for exploratory and comparative housing insights.
